# Tic Tac Toe
A responsive Tic Tac Toe game built with vanilla JavaScript. Play against another local player or the computer.

## Features

- Player vs Player mode
- Player vs Computer mode
- Score tracking
- Responsive layout

## Tech Stack

- HTML
- CSS
- JavaScript
